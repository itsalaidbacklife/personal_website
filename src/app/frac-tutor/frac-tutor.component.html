FracTutor

At any given moment, some students need worked examples to model how to perform a task, while others are ready for problem solving practice. How can teachers balance these diverse needs?

FracTutor personalizes instruction by giving each student an interleaved blend of problem solving practice and worked examples with self-explanation prompts, based on the student's ability as measured by Bayesian Knowledge Tracing.

FracTutor is a Cognitive Tutoring System built using the Cognitive Tutor Authoring Tools (CTAT) developed at Carnegie Mellon University, as well as the nools rule engine. It is designed to teach middle school students to compare fractions using four different strategies.

When a problem loads, the tutor evaluates the student's profficiency at applying the applicable strategies, and provides worked examples or problem solving practice to suit the student's needs.
