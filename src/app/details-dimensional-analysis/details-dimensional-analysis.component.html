DoseTutor

How much help do students need? With what level of granularity should we break down students' homework into steps? How can this granularity be optimized for each student?

DoseTutor addresses these challenges with dynamically-scaled scaffolding that breaks steps into substeps when students struggle.

DoseTutor is an Intelligent Tutoring System built using the Cognitive Tutor Authoring Tools (CTAT) developed at Carnegie Mellon University. It is designed to teach Veterinary Technicians to calculate drug dosages using dimensional analysis. The problem initially assumes students will require minimal assistence and splits the problem into only 4 steps.

If students make mistakes on any of the original 4 steps, they are brought to a separate diaglog that splits the current step into 3-4 substeps. This allows advanced students to answer the question in a few simple steps, while modeling for struggling students how the sequence of information retrieval and calculation required to solve the problem can be broken into reasonably sized chunks.
